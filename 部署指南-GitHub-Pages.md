# Sundays 独立站部署指南 — GitHub Pages（完全免费）

## 总费用：¥0

---

## 第一步：注册 GitHub 账号（5分钟）

1. 打开 [https://github.com/signup](https://github.com/signup)
2. 输入你的邮箱地址（推荐用 sundaysbilliard@hotmail.com）
3. 设置密码
4. 输入用户名（推荐：`sundays-billiard` 或 `sundaysbilliards`）
5. 完成邮箱验证

---

## 第二步：创建仓库（2分钟）

1. 登录 GitHub 后，点击右上角的 **"+"** → **"New repository"**
2. 仓库名填写：`sundays-billiard.github.io`
   - **重要**：仓库名必须是 `你的用户名.github.io`，这样才能直接用这个域名访问
3. 选择 **Public**（公开）
4. 勾选 **"Add a README file"**
5. 点击 **"Create repository"**

---

## 第三步：上传网站文件（3分钟）

1. 进入刚创建的仓库页面
2. 点击 **"Add file"** → **"Upload files"**
3. 把 `sundays-website` 文件夹里的 `index.html` 文件拖拽到上传区域
4. 在下方 "Commit changes" 处填写说明（如 "upload website"）
5. 点击 **"Commit changes"**

---

## 第四步：启用 GitHub Pages（2分钟）

1. 在仓库页面点击 **"Settings"**（设置）
2. 左侧菜单找到 **"Pages"**
3. Source 选择 **"Deploy from a branch"**
4. Branch 选择 **"main"**，文件夹选 **"/ (root)"**
5. 点击 **"Save"**
6. 等待 1-2 分钟，刷新页面

---

## 第五步：访问你的网站！

你的网站地址就是：**https://你的用户名.github.io**

例如：`https://sundays-billiard.github.io`

---

## 常见问题

### Q: 网站打不开？
A: GitHub Pages 部署需要 1-5 分钟，请耐心等待后刷新。

### Q: 怎么修改网站内容？
A: 在 GitHub 仓库里点击 `index.html` → 点击铅笔图标编辑 → 修改后点 "Commit changes"。

### Q: 以后想用自定义域名怎么办？
A: 
1. 购买域名（如在阿里云、Namesilo 购买 `sundaysbilliards.com`，约 ¥50-60/年）
2. 在域名管理后台添加 CNAME 记录，指向 `你的用户名.github.io`
3. 在 GitHub Pages 设置里的 "Custom domain" 填入你的域名
4. 勾选 "Enforce HTTPS"
